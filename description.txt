THIẾT KẾ WEB QUẢN LÝ CHI TIÊU

Mục tiêu
Thiết kế và xây dựng một web app quản lý chi tiêu cá nhân có thể mở rộng nhiều user, với AI tích hợp để hỗ trợ nhập liệu và phân tích.

Yêu cầu tổng thể

Xây dựng một hệ thống gồm:
Frontend: Reactjs (responsive, dashboard chuyên nghiệp)
Backend: Nodejs (Express)
Database: MySQL
Có authentication (JWT hoặc session)
Deploy online (production-ready)

AI Integration 
Feature cần có:
1. Nhập chi tiêu bằng ngôn ngữ tự nhiên

Ví dụ:"Hôm nay ăn trưa 50k"

→ AI phải:
parse:
amount
category
date
gọi API tạo transaction

2. Hỏi đáp báo cáo
Ví dụ: "Tháng này tôi tiêu bao nhiêu cho ăn uống?"
→ AI:
gọi API report
trả lời dạng text

3. Phân tích chi tiêu
So sánh tháng
Detect tăng/giảm bất thường

4. Gợi ý tiết kiệm
Dựa trên pattern chi tiêu
