# Frontend Service

React + Vite UI cho he thong quan ly chi tieu.

## Cai dat

```bash
npm install
```

## Chay local

```bash
npm run dev
```

Frontend chay mac dinh tai `http://localhost:5173`.

## Bien moi truong

- `VITE_BACKEND_URL` (default: http://localhost:4000/api)

## Page structure

- Dashboard
- Transactions
- Wallets
- Categories
- Reports
- Settings

Tat ca API call duoc thong qua `src/services`.
